# Operations Dashboard — Install Guide

## Step 1 — Install Tampermonkey

Click the link for your browser and follow the prompts to add the extension:

- **Chrome:** https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo
- **Edge:** https://microsoftedge.microsoft.com/addons/detail/tampermonkey/iikmkjmpaadaobahmlepeloendndfphd
- **Firefox:** https://addons.mozilla.org/en-US/firefox/addon/tampermonkey/

Once installed you'll see the Tampermonkey icon in your browser toolbar.

## Troubleshooting

**Dashboard not showing up?**
Make sure the Tampermonkey icon is visible in your toolbar and the script shows as "Enabled" in the Tampermonkey dashboard (click the icon → Dashboard).

**Getting a blank page or errors?**
Make sure you're connected to the company network or VPN.
